# Proyecto: Sistema de Gestión de Notas de Estudiantes (POO)

Este es un proyecto universitario desarrollado en Python que implementa un sistema básico para administrar las notas y el estado académico de estudiantes.

## Descripción

Este proyecto destaca por el uso de **Programación Orientada a Objetos (POO)**. Se define una clase `Estudiante` que encapsula todos los datos relevantes (RUT, nombre, materia) y sus funcionalidades (métodos).

El sistema permite registrar nuevos estudiantes, ingresar las 4 notas de un estudiante específico, y luego listar a todos los alumnos mostrando su promedio ponderado y su estado (Aprobado o Reprobado) basado en ese promedio.

## Funcionalidades Principales

* **Programación Orientada a Objetos (POO):**
    * Definición de la clase `Estudiante` con atributos.
    * Métodos como `.ingreso_nota()`, `.notas_promedio()` y `.estado_estudiante()`.
* Registro de múltiples estudiantes (instancias de la clase).
* Cálculo de promedios ponderados.
* Menú interactivo para gestionar el sistema.

## Tecnologías Utilizadas

* **Python:**
    * Programación Orientada a Objetos (Clases, Métodos, Instancias).
    * Estructuras de datos (Listas para almacenar objetos).
    * Funciones.
